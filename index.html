Project SQUOIA

Resources and applications developed for the project SQUOIA ('Exploiting Parallel Treebanks for Hybrid Machine Translation') at the University of Zurich. Contains:

  • Code for hybrid MT-Systems Spanish-Cuzco Quechua and Spanish-German
  • Bilingual dictionaries Spanish-Cuzco Quechua and Spanish-German in Apertium format
  • Parallel treebanks in Spanish-German-Quechua, annotated with phrase structures (German and Spanish) and dependency trees (Quechua)
  • xfst and foma sources for morphological analysis and spell-checking for Cuzco Quechua (spell checker plugins for LibreOffice/OpenOffice are available here)
  • a pipeline for analysing and parsing Southern Quechua texts
